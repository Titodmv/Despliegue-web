# Instalar Nginx
> Actualizar equipo
```
sudo apt-get update
sudo apt-get upgrade
```
> Instalar nodejs
```
sudo apt install nodejs
```
> Comprovar nodejs
```
node -v
```
> Instalar npm
```
sudo apt install npm
```
> Instalar Ngix
```
sudo apt install nginx
```
> Iniciar y habilitar Ngix
```
sudo systemctl start nginx
sudo systemctl enable nginx
```
